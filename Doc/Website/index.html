About

Code12 is the Java Programming system designed for introductory programming education.

Encourage experimentation and independent work with the simple and fun graphics-oriented API, friendly error messages designed with beginners in mind, and instant edit-compile-run response.

Code12 minimizes the complexity of Java and object-oriented concepts to allow students to concentrate on learning the basics of programming.
