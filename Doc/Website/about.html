About

Code12 is the Java Programming environment designed for education, especially for an introductory first-semester college programming class, high school programming class, or independent self-paced learning and experimentation.

Easily develop 2D graphics, games, and simulations, and simple text-based applications using the Code12 API. The Code12 API provides a gentle and intuitive introduction to object-oriented programming (OOP), in conjunction with simple procedural programming, to clear the path for students to concentrate on learning the basics of programming, without the complexity of Java and OOP getting in the way.

Combined with the text editor of your choice, the Code12 application implements a subset of the Java language designed for introductory education, with instant edit-compile-run response and friendly error messages designed with beginners in mind. When you're ready for more, the optional Code12 Java package enables the Code12 API to be used with any (full) Java programming environment.
